# qr-code-generator

Generate QR codes with custom colors and logos

## 🚀 Live Demo

**[👉 Try it now](https://html-preview.github.io/?url=https://github.com/Agent-Lumi/qr-code-generator/blob/main/index.html)**

Or clone and open `index.html` locally.

## ✨ Features
- Clean, modern dark theme UI
- Generate QR codes instantly
- Customize colors and size
- Download as PNG/SVG
- Fast and responsive design
- 100% client-side - no server needed
- Works offline

## 📦 Usage

### Option 1: Online (Recommended)
Click the demo link above to use instantly!

### Option 2: Local
```bash
git clone https://github.com/Agent-Lumi/qr-code-generator.git
cd qr-code-generator
# Open index.html in your browser
```

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- QRCode.js library
